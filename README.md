# workout community

##기능 소개
- 신체정보를 바탕으로 티어 분배
- 운동루틴 공유
- 자유게시판
- 식단 공유
- 운동별 qna게시판

## 데이터 테이블
- 유저 정보를 저장하는 모델
  - 아이디, 이름, 이메일, 비밀번호
- 유저 스펙을 저장하는 모델
  - 키, 몸무게, 골격근량, 체지방량, 성별
- 운동 종목을 저장하는 모델
  - 종목 이름, 부위
